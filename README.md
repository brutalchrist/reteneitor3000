## Dependencies

```
# apt-get install qt6-base-dev qt6-base-dev-tools festival festvox-ellpc11k sox libsox-fmt-mp3 vorbis-tools cmake build-essential
```


## Compile

```
$ cd reteneitor3000
$ mkdir build
$ cd build
$ cmake ..
$ make
```


## Run

```$ ./reteneitor3000```
